
## Getting Started

Descripción:

Este proyecto implementa una lista simple en C++, utilizando clases para permitir listas de diferentes tipos de datos. Se basa en el paradigma de programación orientado a objetos (OOP), ya que encapsula la estructura de los nodos y la lista en clases con métodos específicos.

## Requirements

Compilador de C++ como el MinGW
Sistema operativo: Windows

Con el archivo MinGw se encuentra una carpeta bin, se recomienda copiar la ruta de esta carpeta en el path, para poder ejecutar el programa facilmente, para esto, se tiene que copiar Win + R y escribir sysdm.cpl, despues acceder a "opciones avanzadas" y dar clic en "Variables de Entorno", a continuacion en "Variables del Sistema" buscar el apartado path, seleccionarlo y agregar en "Nuevo" la ruta donde se encuentra el bin del archivo MinGW. Esto permitira ejecutar los programas de forma mas sencilla.

## Form Of Execution

Teniendo en cuenta que ya se agrego la ruta del bin en el path del sistema, accedemos al cmd, nos tenemos que ubicar en la ruta donde se encuentra el archivo main, ya ubicados en la ruta del archivo se procede a escribir (g++ Main.c++ -o programa.exe) y en seguida escribimos (programa.exe) "En Windows". y se observara la ejecucion del programa.
